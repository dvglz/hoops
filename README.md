# HOOPS

Mobile-first Three.js basketball shootaround built with `vite`, `typescript`, and `three`.

## Run Locally

Install dependencies:

```bash
npm install
```

Start the local dev server:

```bash
npm run dev
```

Vite will print the local URL in the terminal, usually `http://localhost:5173/`.

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Notes

- The game is designed for portrait/mobile play first.
- There is a built-in debug menu in the top-right for `gravity`, `ambience`, and `ball skin`.
